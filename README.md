# Master

## Setup:
1. Create virtual environment: `python -m venv venv`
2. Activate virtual environment: `venv\Scripts\Activate.ps1` (Windows PowerShell)
3. Install packages: `pip install -r requirements.txt`

## Useful commands:
- Put all installed packages in requirements.txt: `pip freeze > requirements.txt`
- Deactivate virtual environment: `deactivate`
